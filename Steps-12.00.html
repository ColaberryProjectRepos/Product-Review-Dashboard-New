Data Transformation 12



In this step you will clean and transform the data:

  1.  Select Transform Data from Home tab
  2.   Rename Table: Change the table name to `ElectronicProducts`.
  3.  Delete Unnecessary Columns: Remove the following columns: `asins`, `dimension`, `ean`, `ImageURLS`, `Keys`, `reviews.numHelpful`, `reviews.sourceURLs`, `SourceURLs`, `upc`, and `weight`.
  4. Manufacturer Column: Replace null values in the `manufacturer` column with "Not Specified".
  5.  Change Date Type: Change the data type of the `reviews. date` column to `Date`.
  6.  DoRecommend Column: Replace null values in the `reviews.doRecommend` column with `TRUE`.
  7.  Filter Ratings Column: Deselect null values in the `reviews. rating` column.
  8.  Rename the Name column with Products
