Adding Project Title and Companies Logo 16



1. In the report view, Select Insert ribbon>Text box

a. Type the Project Name ‘Product Review Dashboard’

b. Highlight the Text and adjust Size = 32, Font = Segoe UI, Center ,Color=#4184C5.

c. In the Formatting Tab-->Size and Style: Height= 82, Width = 1278, Position: Horizontal = 1, Vertical = 0, Effect: Background: On, Color = White, Transparency =0%

2. In the Report View, Select Insert ribbon > Image > and select the Company logo.

a. In the Formatting Tab-->Size and Style: Height = 83, Width = 199, Position: Horizontal = 0, Vertical = 0, Background: ON.
