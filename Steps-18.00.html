Add a Date Year Slicer and Brand Slicer 18



1. Adding Year Slicer:

-  In the Report View, open the visualization pane, then Select Slicer from the visualization
 - In the fields, open the ‘Calendar’ Table, and drag the Year.
-  In the Formatting Tab--> Size and Style: Height = 104, Width 217, Position: Horizontal = 0, Vertical = 82 ,Background color=ON, Border=White
Title: Year, Font = Segoe UI, Size =12, Horizontal alignment= Center, Text color = #FFFFFF , Background Color =#4184C5
- Slicer settings: Options: Between
Slicer Header: OFF
-Callout Values: Font = Segoe UI , Size = 10, Color: #252423 ,Slider: On
